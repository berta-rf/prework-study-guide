HTML

  • The head element contains information about the webpage (metadata, not visible to user).
  • The body element represents the visible content shown to the user.
  • With visible HTML elements, the content is contained between an opening tag and a closing tag (angle brackets with associated tag names contained inside - example: <head>content</head>). We insert the content between an opening tag and a closing tag so that the browser understands how to treat the content.
  • The <title> element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When using a class attribute as selector we can assign any CSS rule to selected HTML elements.
  • A CSS rule is formed by selector + declaration block (one or many declarations of property+value -> ex.'color: blue;').

Git

  • git status: checks what branch we are currently on.
  • git checkout "branch-name": switches your location to other branch
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add: adds modifications in current working branch to staging area (either list items one by one or git add -A: adds all changes).
  • git commit -m "COMMIT MESSAGE": adds a description of what the commit contains.
  • git pull origin main: good to check we have latest version of the project in our local computer.
  • git push origin "branch-name": pushes the changes made locally to the remote GitHub branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditionals are code structures used to test if an expression returns true or not. A very common form of conditionals is the if...else statement.
  • Functions are a way of packaging functionality that you wish to reuse. When you define a body of code as a function it will execute when you call the function name in your code.
